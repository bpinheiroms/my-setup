# my-setup

Minimal portable setup for my OpenCode configuration.

## Included

- `opencode/`
  - OpenCode config
  - Oh My OpenAgent routing for OpenCode Go
  - docs: [`opencode/README.md`](opencode/README.md)

## Principles

- Keep only reusable OpenCode setup.
- OpenCode Go orchestration is owned by Oh My OpenAgent.
- Keep secrets, tokens, histories, caches, sessions, and logs out of version control.
