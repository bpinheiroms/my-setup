# Zellij

Zellij configuration focused on fast pane/tab navigation and a reusable base setup.

## Files

- `config.kdl`
  - redefines the default shortcuts (`clear-defaults=true`)
  - creates a workflow closer to personal muscle memory than the Zellij preset

## What Is Intentionally Not In This Repo

Project-specific layouts were removed from this repository.

Reason:

- they are too tied to local paths and current projects
- they change often
- the reusable value here is the base interaction model, not the project tabs themselves

## Main Shortcuts

### Normal Mode

- `Ctrl-a`: previous tab
- `Ctrl-s`: next tab
- `Ctrl-d`: horizontal split downward
- `Ctrl-e`: vertical split to the right
- `Ctrl-f`: toggle fullscreen for the focused pane
- `Ctrl-x`: next pane
- `Ctrl-z`: previous pane
- `Alt-s`: enter scroll mode

### Pane Mode (`Ctrl-p`)

- `h j k l` or arrows: move focus between panes
- `d`: new pane below
- `r`: new pane to the right
- `n`: new pane with the default layout
- `s`: new stacked pane
- `w`: show/hide floating panes
- `f`: fullscreen
- `i`: pin/unpin da pane
- `i`: pin/unpin the pane
- `z`: show/hide frames

### Tab Mode (`Ctrl-t`)

- `h` / `k`: previous tab
- `j` / `l`: next tab
- `n`: new tab
- `x`: close tab
- `r`: rename tab
- `s`: sync the active tab
- `1..9`: jump directly to a specific tab

### Auxiliary Modes

- `Ctrl-n`: resize mode
- `Ctrl-h`: move mode
- `Ctrl-o`: session mode
- `Ctrl-b`: tmux mode
- `Ctrl-g`: lock/unlock

## Notes

- The current `config.kdl` was generated by Zellij itself and kept a backup as `config.kdl.bak` on the local machine.
- The focus here is productivity through muscle memory, not strict adherence to the default Zellij preset.
- Project layouts can still exist locally, but they are intentionally not versioned here.
